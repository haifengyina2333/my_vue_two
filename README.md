# 项目说明
一个基于 Vue 2.7 和 Element-ui 的后台管理项目
## 项目相关依赖安装
```
npm install
```

### 运行项目
```
npm run serve
```

### 使用开发者模式运行项目
```
npm run build
```

### 进行代码规范检查和修复
```
npm run lint
```
